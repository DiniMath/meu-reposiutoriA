Célula animal

Os diferentes tipos de seres vivos tem celulas e organismos diferentes entre-si.Aqui será apresentado a célula animal.

Organelas e estruturas

A célula animal apresenta diferentes tipos de organelas e estruturas,que desepenham funçoes complementares entre si para se manterem vivas.Além de outros elementos importantes para o seu funcionamento como o hialoplasma.

As principais organelas abordadas serão:

  • mitocôndria
  • Complexo de Golgi
  • Ribossomo
  • Lisossomo
  • Peroxissomo
  • Retículo endoplasmático liso
  • Retículo endoplasmático rugoso

As principais estruturas abordadas serão:

  • Núcleo
  • Citoesqueleto
  • Membrana Plasmática
